Complete interactive troubleshooting guide for power-related issues

Does the computer show any signs of power?

Check for LED lights, fan movement, or audible sounds when attempting to power on

Technical Note: Some systems may have lights but no fan movement, or vice versa.

Is there display output on your monitor?

Check if you see anything on screen (BIOS, OS loading, etc.)

Path: System has power →

Does the system boot completely to the operating system?

Check if the boot process completes successfully

Path: System has power → Display works →

System appears to be working normally

If you're experiencing intermittent issues, consider these checks:

  1. Monitor system temperatures
  2. Check for loose connections
  3. Test with different power outlet
Note: Some power issues may be related to overheating or unstable power supply.
Path: System has power → Display works → Boots completely

Is the power supply receiving electricity?

Check these potential issues:

  • Is the power cable properly connected?
  • Is the wall outlet working?
  • Is the PSU switch turned on?
Path: No power →

Power Supply Not Receiving Electricity

Try these solutions:

  1. Test with a different power cable
  2. Try a different wall outlet
  3. Check circuit breakers/power strips
  4. Test the outlet with another device
Advanced Check: Use a multimeter to verify outlet voltage (should be ~120V or ~240V depending on region).
Path: No power → PSU not receiving electricity

System shuts off randomly

Possible causes:

  • Overheating
  • Power supply failure
  • Motherboard issues
Path: Intermittent → Random shutdowns

Check System Temperatures

Use BIOS or software tools to monitor CPU, GPU, and system temperatures.

Clean dust from fans and heatsinks.

Path: Intermittent → Random shutdowns → Check temperatures

Test with Different Power Supply

If possible, test with a known good PSU of adequate wattage.

Path: Intermittent → Random shutdowns → Test PSU

Intermittent Power On

Check power connections and try different power source.

Path: Intermittent → Sometimes doesn't turn on

Unexpected Reboots

Check for driver issues, software conflicts, and RAM problems.

Path: Intermittent → Unexpected reboots

Check Video Connections

Ensure monitor cable is properly connected, test with another cable if available.

Path: System has power → No display

Boot Process Failure

Check boot order in BIOS, test hard drive, try reinstalling OS.

Path: System has power → Display works → Boot failure

Inconsistent Booting

Check hard drive health, test RAM, and check for loose connections.

Path: System has power → Display works → Inconsistent boot

Power Supply Receiving Electricity

Proceed to motherboard or PSU troubleshooting.

Path: No power → PSU receiving electricity
